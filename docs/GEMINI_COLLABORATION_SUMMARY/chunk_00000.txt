## TROUBLESHOOTING WITH GEMINI

### If Commands Fail:

**1. Copy Exact Error Message**
```
Tell Gemini: "I got this error when running the command:"
[Paste full error message]
```

**2. Provide Context**
```
Tell Gemini: "I was on step [X] when this happened"
"The previous step completed successfully"
"I'm running as user: [username]"
```

**3. Request Diagnosis**
```
Ask Gemini: "What's causing this error?" "How can I fix this?" "Is there an alternative approach?" ```

**4. Implement Solution**
```
Execute: Gemini's corrected command
Report: "✓ That worked" or "✗ Still getting error"
Continue: Next troubleshooting step
```

### Common Issues Gemini Can Solve:

- Excel file locked or in use
- Insufficient permissions
- Schema mismatch on import
- Feature service connection timeout
- Broken data sources after update
- Performance issues with large dataset
- Null value handling
- Date format conversion
- Geodatabase locks
