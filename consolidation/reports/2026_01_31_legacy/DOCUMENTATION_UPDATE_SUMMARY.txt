VERIFICATION COMMANDS PROVIDED
=============================================================================

After Python module extraction, users can run:

1. Install dependencies:
   pip install -r requirements.txt

2. Test imports:
   python -c "from shared.utils.schema_loader import ConfigLoader"
   python -c "from shared.processors.field_normalizer import FieldNormalizer"
   python -c "from shared.validators.validation_engine import ValidationEngine"
   python -c "from shared.validators.quality_scorer import QualityScorer"

3. Validate schema paths:
   python -m shared.utils.schema_loader

4. Test configuration loading:
   python -c "from shared.utils.schema_loader import load_config; print(load_config('schemas.yaml'))"

5. Run dry-run:
   python run_consolidation.py --dry-run --verbose

6. Run validation-only:
   python run_consolidation.py --validate-only --verbose

7. Use Makefile:
   make help
   make init-dirs
   make check-schemas
