=============================================================================
TROUBLESHOOTING
=============================================================================

Issue: "File not found" error during consolidation
Fix: Check that all source files exist in 05_EXPORTS\_CAD
     Verify paths in consolidate_cad_2019_2026.py

Issue: "Memory error" during processing
Fix: Close other applications
     Process in chunks (modify script if needed)
     Ensure 8GB+ RAM available

Issue: Low quality score (<99)
Fix: Check normalization report for issues
     Review validation output for domain violations
     Re-run if temporary I/O issue

Issue: Missing RMS data during backfill
Fix: Verify RMS files in CAD_Data_Cleaning_Engine/data/rms/
     Copy from 05_EXPORTS\_RMS/csv/yearly/ if needed

Issue: ChromaDB or pydantic errors
Fix: These are unrelated to CAD consolidation
     CAD pipeline doesn't use ChromaDB
     Ignore if running CAD consolidation only
